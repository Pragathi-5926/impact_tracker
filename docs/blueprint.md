# **App Name**: CampusConnect SDG

## Core Features:

- Role-Based Authentication: Firebase Authentication with email/password and Google Sign-In for different roles (Admin, Staff, Student).
- Admin Dashboard: Dedicated dashboard for Admin to manage members (HoDs), view department reports, and manage settings & permissions using database tools.
- Staff Dashboard: Dashboard for Staff to add student activities, verify submissions, and view department analytics.
- Student Dashboard: Dashboard for Students to track personal SDG progress, submit new activities, and view leaderboard rankings.
- Activity Submission: Students can submit new SDG-related activities with detailed descriptions, and links for documentation.
- Submission Verification Tool: Staff members use AI tool to verify and approve/reject student submissions based on credibility of provided links to documentation.
- Analytics and Reporting: Generate department-wise analytics and reports for Admin and department-specific analytics for Staff. Includes Firestore data structure.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) for trustworthiness and stability.
- Background color: Light Gray (#F0F4F7) to provide a clean, modern feel.
- Accent color: Soft Green (#8BC34A) to highlight SDG-related activities.
- Body and headline font: 'Inter' sans-serif for a modern, objective feel.
- Use simple, line-based icons related to each dashboard section and activity type.
- Responsive layout adapting to different screen sizes, ensuring mobile-friendliness.
- Subtle transition animations when navigating between dashboard tabs.
